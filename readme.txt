== Description ==

The SMS Media4u Login plugin adds the functionality to send SMS by use of Media4u in order to login WordPress.

= About Media SMS =

Media SMS is a message delivery system developed by Media4u.

[Media SMS](https://media-sms.net/)

In order to use this plugin, you have to purchase the authentication key at the following site.

[WordPress Market](https://www.wpmarket.jp/)

== Installation ==

1. Copy the `sms-media4u-login` directory into your `wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! :)
